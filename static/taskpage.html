Задание для практики с циклом While

By : Ana Mayer

У Алисы есть коробка с маленькими квадратами 1х1 внутри. Она любит делать квадраты с ними на столе. Она хочет, чтобы вы помогли ей выяснить, сколько квадратов разных размеров она может сделать с маленькими квадратами в своей коробке.

Входные данные:

Количество маленьких квадратов внутри коробки.

Вывод:

Количество разных квадратов она может сделать.

Пример:

Вход: 50

Выход: 7

Решение

Если мы посмотрим на пример, мы увидим, что Алиса имеет 50 маленьких квадратов. Таким образом, она может сделать следующие квадраты:

1x1 = 1, 2x2 = 4, 3x3 = 9, 4x4 = 16, 5x5 = 25, 6x6 = 36, 7x7 = 49.

Мы можем использовать счетчик и цикл while, где счетчик будет представлять длину стороны квадрата. Итак, нам нужно просмотреть все значения c от одного до тех пор, пока квадрат c не станет больше, чем количество маленьких квадратов в коробке. (физически мы не можем сделать квадрат):

n = int(input())

c = 1

answer = ""

while c*c < n:

answer += str(c*c) + " "

c += 1

print(answer)
